Stage 2 data folder

Please place your Stage 2 train/test files in this folder using the following names:
- train.txt
- test.txt

Expected format (one sample per line):
feature_1 feature_2 ... feature_n label

Notes:
- Whitespace-separated and comma-separated rows are both supported.
- The last value in each row must be the class label.
- Labels should be integer class ids (for example: 0,1,2,...).
